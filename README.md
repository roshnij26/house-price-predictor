# house-price-pridictor
Machine Learning Regeression model for predicting house prices using historical property data.

## Dataset
- Source: historical property Dataset for Regression
- Shape: 2919 rows x 13 columns

## Technologies Used
- Python
- pandas
- NumPy
- scikit-learn

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis(EDA)
3. Data Preprocessing
4. Model Training
5. Evaluation

## Results
### Regression
- Mean Absolute Error(MSE): 30829.93
- R2 Score: 0.374
